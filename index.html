个人博客的搭建

具体步骤如下

使用Hexo搭建博客，基于windows静态博客搭建

  1. 下载node.js,安装地址:https://nodejs.org/en/

  2. 安装完之后再cmd窗口查看node和npm版本,node -v,npm -v

  3. 安装之前要借助npm包管理器，国内由于镜像源的速度很慢，所以要利用npm来安装cnpm：npm install -g cnpm –registry=https://registry.npm.taobao.org

  4. 查看版本 cnpm -v

  5. 利用cnpm安装Hexo,cnpm install -g hexo-cli,至此，Hexo博客就安装成功 ,用hexo -v验证

  6. 建立文件夹，mkdir blog,所有的东西将会在blog中，如果搭建博客出现了问题，打不了把文件夹删除，重新搭建，进入blog ，cd blog

  7. hexo init(在windows平台，要先安装git，初始化才能生效)

  8. dir查看目录，启动博客，hexo s,此时在本地的localhost:4000端口来启动，打开浏览器访问

  9. hexo n “文章名”;cd blog/source/_posts

  10. cd ../../,到blog目录；hexo clear(清理一下);hexo g(生成);hexo s(启动）

  11. 可以将博客部署到github上，可以通过github地址来访问博客，这样会比较好，利用cnpm安装一个git部署的插件，安装方式：cnpm install –save hexo-deployer-git

  12. 设置_config.yml，在记事本中找到最后，x修改

  13. type: git

    repo: https://github.com/645575958/645575958.github.io.git

    branch: master

  14. 部署到远端，命令为:hexo d,再次之前应在blog下输入：git config –global user.email “645575958@qq.com“

git config –global user.name “645575958”

搭建博客遇到的问题

  • 第一个坑：hexo init,我的电脑没有安装git ,要先安装git,然后在进行其他的操作，中途安装也要重新开始
  • 第二个坑，将博客部署到github上，在blog，输入git config –global user.email “emali地址”；it config –global user.name “用户名”
